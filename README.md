# Tez-Calismalari-Preview

Things I want to serve in GitHub Pages.

## How to Use

1. Add your HTML files to the `docs/` folder
2. Push your changes to the `main` branch
3. GitHub Actions will automatically deploy your files to GitHub Pages

## Setup

To enable GitHub Pages, go to your repository **Settings** > **Pages** and set:
- **Source**: GitHub Actions

Your site will be available at: `https://<username>.github.io/Tez-Calismalari-Preview/`
